configure

Name

configure -- Change global gnocl settings.

Synopsis

configure [-option value...]

Description

With the configure command global gnocl settings can be changed.

Options

-defaultIcon

type: PERCENTSTRING

Sets the default icon for windows whose icon is not explicitely set. This must be either a file name (prefix "%/") or empty.

-tooltip

type: boolean

Set globaly if the tooltips are shown.
